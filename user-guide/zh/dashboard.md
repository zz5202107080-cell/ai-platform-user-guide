# 控制台使用指南

AI大模型聚合平台 控制台提供了完整的管理界面，方便用户配置和管理 AI 服务。

## 📊 控制台概览

登录后，您将看到以下主要功能区域：

### 左侧导航菜单
- **对话** - 操练场测试模型和聊天
- **控制台** - 数据看板，令牌管理和使用日志
  - **数据看板** - 系统概览和关键指标
  - **令牌管理** - API 密钥创建和管理
  - **使用日志** - API 调用记录查询
- **个人中心** - 钱包管理和个人设置

## 🏠 数据看板（Dashboard）

数据看板提供系统关键指标的概览视图。

### 1. 统计数据卡片
- **账户数据** - 账户当前余额和历史消耗
- **使用统计** 
  - 请求次数：账户总请求次数
  - 统计次数：当前查询时间范围内的请求次数
- **资源消耗** 
  - 统计额度：当前查询时间范围内消耗的额度
  - 统计Tokens：当前查询时间范围内消耗的Token数量
- **性能指标** 
  - 平均RPM：Requests Per Minute，每分钟请求数
  - 平均TPM：Tokens Per Minute，每分钟Token数。


### 2. 模型数据分析
- **消耗分布** - 模型使用消耗分布
- **消耗趋势** - 最近模型的调用量变化
- **调用次数分布** - 各 AI 模型的使用比例
- **调用次数排行** - 最常用的 AI 模型排行

## 🔑 令牌管理（Token Management）

### 令牌列表视图
- 显示所有已创建的 API 令牌
- 每行显示：令牌名称、令牌状态、剩余额度/总额度、密钥（部分隐藏）、可用模型、IP限制、创建时间、过期时间
- 支持按名称搜索、密钥搜索

### 创建新令牌
点击"添加令牌"按钮，填写表单：

| 字段   | 说明              | 必填   |
|------|-----------------|------|
| 令牌名称 | 易于识别的名称         | 是    |
| 过期时间 | 设置令牌有效期         | 是    |
| 新建数量 | 设置令牌数量          | 是    |
| 额度设置 | 设置令牌可用额度        | 否    |
| 访问权限 | 选择该令牌可访问的 AI 模型 | 否    |

### 令牌操作
- **编辑** - 修改令牌配置（名称、模型权限等）
- **复制密钥** - 复制完整的 API 密钥
- **禁用/启用** - 临时停用或重新启用令牌
- **删除** - 永久删除令牌（谨慎操作）

## 📝 使用日志（Usage Logs）

### 日志查询
- **时间筛选** - 按时间范围查询日志
- **条件筛选** - 按用户、令牌、模型、状态等筛选

### 日志详情
每条日志记录包含：
- **时间戳** - 请求发生的时间
- **用户/令牌** - 发起请求的用户或令牌
- **模型** - 请求的 AI 模型
- **渠道** - 实际使用的 AI 提供商渠道
- **状态码** - HTTP 状态码和错误信息
- **耗时** - 请求处理时间
- **输入/输出** - 请求和响应内容（可展开查看）

## ⚙️ 个人中心（Personal Setting）

### 个人设置
- **账户绑定** - 绑定、修改邮箱
- **安全设置**
  - 系统访问令牌：生成用于API调用的身份验证令牌
  - 密码管理：修改账户登录密码
  - Passkey登录（无密码认证）：基于WebAuthn标准的现代无密码认证
  - 两步验证设置：完整的TOTP（基于时间的一次性密码）两步验证系统
  - 删除账户：永久删除用户账户
- **通知设置** - 配置接收的通知类型


## 🆘 常见问题

### Q: 令牌剩余额度不准确？
A: 检查是否有未完成的异步请求，或等待统计系统更新（通常有几分钟延迟）。

### Q: 统计图表数据延迟？
A: 统计数据通常有5-15分钟的延迟，以确保数据准确性。

### Q: 日志查询太慢？
A: 尝试缩小时间范围或添加更多筛选条件。
